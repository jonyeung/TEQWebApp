Help Menu

Below you can find instructions on how to navigate the website.

Just click on a section below to get help for that function.

Dashboard

The dashboard is the home page of the website, which you first see on login. It shows all the available options that you can use.

To navigate back to the dashboard from another page, you can click on the website's title at the top left of the each page.

You can logout of your account by clicking "Logout" at the bottom of the dashboard.

Uploading an iCARE Template

From the dashboard, click on the "Upload an iCARE Template" button.

Once you are redirected to the upload page, pick the type of iCARE template that you are uploading from the dropdown list (this type refers to the last line of the title on the iCARE Template). Then click "Choose file" to pick the corresponding csv file from your computer that you wish to upload.

Once you are sure that the file specified is the one you want to upload then you may click the "Upload" button on the page to upload the template to the system.

Note: Only one file can be uploaded at a time. If you have multiple iCARE Templates to submit, then upload them separately by repeating the same steps above.

Querying the Data

From the dashboard, click on the "Query the Data" button.

You will be redirected to the query page.

From here you can query the data to see certain columns from the iCARE Templates.

You can select a preset query from the "Chose From Preset Queries" dropdown menu. Once selected, it will show you the column filters that will be applied with this query.

To view the table created using the filters found under "Selected Filters", you can click "Apply Filters". The corresponding table will appear at the bottom of the page.

To clear the filters and start a new query, you can click "Clear Filters".

Querying the Data

From the dashboard, click on the "Query the Data" button.

You will be redirected to the query page.

From here you can query the data to see certain columns from the iCARE Templates.

You can select a preset query from the "Chose From Preset Queries" dropdown menu. Once selected, it will show you the column filters that will be applied with this query.

In addition to this you can add your own column filters.

To add a column filter, you can select a letter that the column header starts with and choose it from the list, it will then be displayed under "Selected Filters". You can also search for a filter using the search bar at the top, select the filter from the suggested list, and then click "Add Filter" to add it to the Selected Filters list.

To view the table created using the filters found under "Selected Filters", you can click "Apply Filters". The corresponding table will appear at the bottom of the page.

To clear the filters and start a new query, you can click "Clear Filters".

Creating New Preset Queries

From the "Query the Data" page, you can create a query and save it so that you may use it again.

To do this, first create a query (see Querying the Data above). Then click "Save Query", which will make a popup window appear. From this window, enter a name for the query in the "Query Name" field, then click "Save".

Now the query is saved and is accessible from the "Chose From Preset Queries" dropdown menu.

Registering a New Account

To create a new account for a user, click "Register a New User" on the dashboard.

Choose a username and password for the new account, then select the user type from the "User Type" dropdown (this decides the access level of this new account).

Click "Register" to submit the new account.

If it successfully registers, then this account can now be used to login.

Change User Permissions

Clicking on "Change User Permissions" at the dashboard will take you to a page where you can view all accounts in the system.

To modify the access level for a particular user, find their row on the "User Information List" and change their user type from the dropdown menu under "Modify Access Level".

Once you have finished selecting the user type for the user(s), click "Save Changes" at the bottom right of the page to save these changes to their account(s).
